## Description

[foosball](http://foosball.zero-team.pro/) raiting tracker.

## Backend

### Setup

```bash
# development
$ ./deploy/scripts/build-dev.sh
$ yarn install
```

### Running

```bash
# development
$ yarn start:dev
```

## Frontend

### Setup

```bash
# development
$ cd frontend/
$ yarn install
```

### Running

```bash
# development
$ yarn dev
```
